ntunhs.contents Installation
----------------------------

To install ntunhs.contents using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``ntunhs.contents`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        ntunhs.contents
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
